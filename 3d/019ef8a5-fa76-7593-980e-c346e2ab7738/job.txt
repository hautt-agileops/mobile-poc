Need someone who can create a 3D print file for a prototype for a two-soda-can holder.
Reference: CruiseCup (https://www.instagram.com/cruisecup/ / cruisecup.com)

How it works (studied):
- Holds two standard 12oz cans stacked vertically inside an insulated tube.
- Slide-to-reload: drink top can, push empty down, spring follower pops the
  lower can up into drinking position; empty ejects like a shell casing.
- Two screw-on lids: top (drinking opening) + bottom (loading).
- Double-wall insulated, 3D printed.

Deliverable: design the inner functional HOLDER core so it can be sent to the
modeler, who adds the outer shell design with logo and branding.

Key dims: 12oz can = Ø66mm x H122mm. Tube ID ~67mm (tight tolerance, Coors
Light/Keystone wider and excluded). Overall H ~265mm. Top + bottom screw lids.
Spring-loaded follower for reload/eject.
